# Stat61 Final Project
by Anna Jing and Alicia Liu

This is an interactive lab that explores creating confidence intervals for QQ plots with generated data and real datasets as well. In this repository, you will find the lab, the code to generate data, necessary CSV data files, and a document with answers. 

1. Final_Project.qmd : Lab file
2. data_examples.csv : Data file
3. data_generation.r : File used to generate data in distTest.csv and mystery_data.csv
4. distTest.csv : Data file
5. heights.csv : Data file
6. mystery_data.csv : Data file
7. sampled_CPS.csv : Data file
